# Sentinel-2 NDVI Change Mapping (Nebraska) — Python + STAC + GeoTIFF

This project builds an end-to-end remote sensing workflow to:
1. Query Sentinel-2 L2A imagery using a STAC API (Microsoft Planetary Computer)
2. Create cloud-filtered NDVI composites for two time windows
3. Compute NDVI change (Time 2 − Time 1)
4. Export GeoTIFF outputs and visualize results on a web map
